# 🚀 什么是 FC-UniApp？
FC-UniApp 是一款基于 `Vite + Vue3 + TypeScript` 构建的企业级 UniApp 快速开发框架。深度封装了路由管理、状态管理、API 请求、国际化、样式处理等核心功能，提供开箱即用的工具函数与最佳实践，让开发者能够专注于业务逻辑，大幅提升跨端应用开发效率与代码质量。

## 🛠️ 核心特性
现代化技术栈：Vite 构建 + Vue3 Composition API + TypeScript 类型安全

完整功能集成：Pinia 状态管理、Vue I18n 国际化、UnoCSS 原子化 CSS

企业级架构：模块化设计、目录结构规范、代码分层清晰

开发体验优化：热更新极速响应、类型提示完善、错误边界处理

跨端一致性：统一 API 层，屏蔽各平台差异，确保多端表现一致

## 🎨 灵活的 UI 策略
FC-UniApp 采用 无 UI 侵入设计，不强制绑定特定 UI 组件库。开发者可根据项目需求自由选择

## 🌐 全平台支持
所有 UniApp 支持的平台均得到全面支持，包括：

H5: 网页应用

小程序生态：微信(其它平台未测试)

移动 App：iOS、 Android、鸿蒙Next 原生应用

## 💡 为什么选择 FC-UniApp？
快速启动：3 分钟完成项目初始化，告别重复配置

最佳实践：内置企业级项目架构，避免新手踩坑

持续更新：紧跟 UniApp 与 Vue3 生态演进

轻量灵活：按需引入功能模块，无冗余代码

文档完善：详细使用指南与示例代码，降低学习成本

FC-UniApp 致力于成为 UniApp 开发者的效率倍增器，让跨端开发变得简单、高效、愉悦。
